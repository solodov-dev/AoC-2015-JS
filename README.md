# Advent of code 2015

## JavaScript

To run JavaScript solutions:

```sh
$ cd js
$ cat ../input/day_1 | node 1-1.js
```

## Rust

To run Rust solutions:

```sh
$ cd rust
$ cat ../input/day_1 | cargo run --bin 1-1
```

## Haskell

To run Rust solutions:

```sh
$ cd haskell
$ cat ../input/day_1 | cabal run 1-1
```
